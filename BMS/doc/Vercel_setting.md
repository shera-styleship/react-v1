##  Vercel 배포

> 배포전 npm run build (리액트 빌드) 굳이 안해도 됨.

> vercel 에서 프로젝트 감지하고 자동으로 빌드

1.  전역 설치
    
    `npm i -g vercel` 
    
2.  로그인
    
    `vercel login` 
    
3.  프로젝트 폴더에서 실행
    
    `vercel` 
    
    -   질문들이 나오면 Enter로 기본값 진행
        
4.  배포
    
    `vercel --prod`


5. 웹 확인
https://vercel.com/home

> Dashboard 에 들어가 생성된 프로젝트 확인
